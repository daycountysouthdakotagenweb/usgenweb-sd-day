Pierpont SURNAMES Beginning with N

The following table is an index of surnames listed in the "Pierpont Centennial History Book: 1887-1987" published in 1987 by The Langford Bugle, Langford, SD. It attempts to list every individual named in the Contribution section of this book. The entire database contains 5412 entries. Unfortunately, in some cases, the book did not list the maiden name of the spouse which precluded their inclusion. When the "Given Name" is noted with a '?,' none was offered in the text.

Indexed by: Curtis P. Sigdestad
